# TalentTrack – Recruitment Landing Page

TalentTrack is a responsive recruitment platform landing page built as a personal frontend project. It showcases a modern hiring workflow, candidate pipeline visuals, animated sections, and a polished business-style UI.

## Features

- Responsive landing page design
- Hero section with video background
- Hiring process cards
- Sticky feature sections
- Candidate pipeline/dashboard UI concept
- Integrations marquee
- Clean footer and rebranded content

## Tech Stack

- HTML5
- CSS3
- JavaScript
- GSAP + ScrollTrigger

## Deployment

This is a static frontend project and can be deployed directly on Vercel, Netlify, GitHub Pages, or any static hosting platform.

### Run locally

Open `index.html` directly in your browser, or use a local server:

```bash
npx serve .
```

## Notes

This project has been rebranded and cleaned for portfolio use. It does not contain client/company credits or original third-party branding in the visible UI.
